# The Laureate Equation

A premium, open, data-driven archive of major mathematics and statistics awards, their laureates, and the ideas that shaped modern mathematics.

## Features

- Award archive
- Laureate profiles
- Multi-level mathematical explanations
- Search and filtering
- Timeline
- Research-field explorer
- World map
- Comparison tools
- Sources and methodology
- Transparent data-completeness status
