# Design System

## Personality

Luxurious, scholarly, timeless, focused, editorial, and internationally credible.

## Visual References

Aim for the quality of leading:

- Museums
- Academic institutions
- Premium editorial publications
- Cultural archives
- Research libraries

Do not imitate a specific website directly.

## Palette

Suggested design tokens:

- Obsidian: `#111315`
- Deep navy: `#111827`
- Charcoal: `#25282C`
- Warm ivory: `#F6F0E5`
- Parchment: `#EEE4D2`
- Antique gold: `#B6924A`
- Champagne: `#D4BC83`
- Muted bronze: `#8F6E3B`
- Body ink: `#202226`
- Muted text: `#666A70`

Gold must be restrained and never look casino-like.

## Typography

Use open-source fonts only.

Preferred:

- Display: Cormorant Garamond, Playfair Display, or EB Garamond
- Body: Inter, Source Sans 3, or IBM Plex Sans
- Mathematics: STIX Two Math or KaTeX-compatible fonts

## Layout

- Strong editorial grid
- Generous whitespace
- Clear heading hierarchy
- Moderate line lengths
- High readability
- Consistent vertical rhythm
- Elegant cards without excessive borders
- Responsive from small mobile to large desktop

## Motion

- Minimal
- Purposeful
- Reduced-motion compliant
- No scroll hijacking
- No autoplay
- No excessive parallax

## Mathematical Motifs

Use subtly:

- Curves
- Geometric constructions
- Coordinate grids
- Topological forms
- Network diagrams
- Manuscript-inspired theorem cards
- Medal silhouettes

Avoid random floating formulas.

## Components

Required:

- Global navigation
- Search
- Breadcrumbs
- Award card
- Laureate card
- Field card
- Source panel
- Completeness badge
- Citation block
- Timeline
- Comparison table
- Filter drawer
- Empty state
- Error state
- Missing image state
- Accessible modal or dialog only when necessary
- Footer

## Themes

Implement light and dark themes. Respect system preference and persist user choice.

## Accessibility

- WCAG 2.2 AA target
- Strong focus states
- Correct heading order
- Minimum contrast
- Keyboard operation
- Descriptive labels
- Touch targets
- Reduced motion
- Text alternatives for all visualisations

## Anti-Patterns

Do not use:

- Generic admin dashboard appearance
- Glassmorphism everywhere
- Neon gradients
- Huge empty hero sections
- Tiny low-contrast text
- Decorative charts without value
- Excessive card grids
- Ambiguous icon-only controls
