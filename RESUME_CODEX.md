# Resume Prompt for Codex

Read `AGENTS.md`, `PROJECT_STATUS.md`, `IMPLEMENTATION_PLAN.md`, and `VALIDATION_LOOP.md`.

Inspect:

- Git status
- Recent commits
- Current project structure
- Existing test results
- The first incomplete milestone
- Any documented blockers or known issues

Resume from the first incomplete milestone recorded in `PROJECT_STATUS.md`.

Do not redo completed, committed, and verified work unless current validation reveals a regression.

Run the relevant validation before continuing. Follow the autonomous loop until full completion or a genuine external blocker. Update `PROJECT_STATUS.md` after each meaningful stage.
