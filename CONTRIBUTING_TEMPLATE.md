# Contributing

## Contribution Types

- Add an award
- Add or correct a winner
- Add a laureate profile
- Improve a mathematical explanation
- Add a verified publication
- Correct a source
- Improve accessibility
- Fix a bug

## Required for Data Contributions

- Stable ID
- Unique slug
- Reliable source
- Access date
- Clear distinction between fact and editorial explanation
- Completeness status
- Last verified date

## Prohibited

- Unsourced claims
- Fabricated quotations
- Copied biographies
- Unlicensed images
- Subjective rankings
- Wikipedia-only sourcing for substantive claims

## Pull Request Checklist

- Data validation passes
- Tests pass
- Links work
- Sources are included
- Image licensing is complete
- No unrelated changes
